# Dot-matrix as archive apparatus

The dot-matrix printer is the first material threshold of the project.

It turns a generative system into impact, rhythm, delay, ink, paper, noise, and residue. It makes computation leave a body.

## Not nostalgia

The printer should not be read as a retro prop. Its force comes from its historical and material function:

- forms;
- invoices;
- ledgers;
- logs;
- tickets;
- reports;
- administrative truth;
- continuous records.

A dot-matrix printer belongs to a world where information is not just displayed. It is registered, carbon-copied, filed, torn, misplaced, archived, and audited.

## Printer as witness

For now, the most useful word is **witness**.

The printer is not only an oracle, because it should not mystify the system. It is not only a bureaucratic machine, because it is also physically fragile and noisy. It is not only an archive machine, because the archive is still being produced.

It is a witness because it records an event as it passes through a machine.

```txt
signal -> generation -> impact -> paper -> archive
```

## Why 9 pins matter

The current prototype is a 9-pin printer. This matters because the mark remains visibly mechanical. The printed line carries artifacts: dotted strokes, noise, uneven density, ribbon wear, registration problems, and texture.

A cleaner 24-pin printer could increase legibility, but might reduce friction. For this project, friction is not a defect by default.

## Sound

The printer sound is a material layer of the work.

It gives time to generation. It refuses the instantaneity of screen output. It makes the system public in the room.

## Slowness

The printer's slowness is a constraint. It limits how much the system can say. That constraint may become an editorial force: the machine must choose what deserves to become paper.

## Accumulation

The output is not a page. It is an accumulation.

Continuous paper can become:

- pile;
- tail;
- ledger;
- dated strip;
- archive box;
- evidence trail;
- unreadable excess;
- public reading surface.

The final spatial behavior remains open.
