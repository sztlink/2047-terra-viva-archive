# Manuals Plus reference map

This is an initial public-safe reference map for the 2047 command center prototype.

Source collection:

- <https://archive.org/details/manualsplus>

Method:

- Search the Internet Archive Manuals Plus collection.
- Select candidate manuals by title, maker, and machine type.
- Link to archive pages only.
- Extract interface grammar, not scanned images.
- Do not copy pages or screenshots into this repository until reuse rights are checked.

Status:

- This is a first shortlist for visual inspection.
- The notes below are hypotheses based on title and equipment type.
- Each item should be opened later and inspected page by page.
