# Manuals Plus reference seeds

This file starts a public-safe reference map for the command center prototype.

The source collection is the Internet Archive Manuals Plus collection:

- <https://archive.org/details/manualsplus>

Do not copy scanned pages into this repository unless reuse rights are checked. Use the collection as a visual corpus for extracting interface grammar.

## Priority visual features

1. Manual page as interface.  
   The command center can behave like a manual page that updates over time.

2. Calibration as score.  
   The sonic score can borrow forms from calibration charts and tolerance tables.

3. Troubleshooting as dramaturgy.  
   Failure modes should appear as structured diagnostic states, not error popups.

4. Foldout as system map.  
   The full apparatus can be represented as a foldout-like diagram.

5. Print proof as receipt.  
   Every smoke test can produce a proof sheet or ledger page.
